## Raspberry Pi Set Up

### Connect the NRF module to the follow GPIO pins

   NRF  |  RasPi
 -----------------------------
   VCC  |   3.3V                       
   GND  |   GND        
   CSN  |   GPIO8  (SPI CE0)         
   CE   |   GPIO17  
   MOSI |   GPIO10 (SPI MOSI)
   SCK  |   GPIO11 (SPI SCLK)
   MISO |   GPIO9  (SPI MISO)
